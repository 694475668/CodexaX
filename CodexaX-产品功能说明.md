# CodexaX —— AI 编程工具 · 产品功能说明

> CodexaX 是一款基于 VS Code（code-oss）深度二次开发的 **AI 原生代码编辑器**，内置自研 AI 助手、智能体（Agent）、代码库语义检索与多模型路由后端，开箱即用、可私有化部署。

---

## 一、产品简介

CodexaX 在 VS Code 内核（code-oss 1.116）之上，重写了 AI 交互层与后端服务，目标是把「对话式编程 + 自动化智能体 + 代码补全 + 代码库理解」整合进一个统一、可控、可计费的工程化平台。

- **完整的编辑器体验**：保留 VS Code 全部能力（多语言、调试、插件、Git、终端、主题等），无学习成本。
- **AI 原生**：聊天、Agent、Tab 补全、代码库检索深度集成进编辑器，不是简单的插件叠加。
- **多模型可切换**：Claude、DeepSeek、GPT、Gemini、xAI、本地 Ollama 一键切换，统一计费。
- **私有化 / 可运营**：自带账号体系、用量计费、管理后台、自动更新服务，可独立部署对外运营。

| 项目 | 说明 |
|---|---|
| 产品名 | CodexaX |
| 内核 | VS Code / code-oss 1.116 |
| 形态 | 桌面客户端（macOS / Windows / Linux） |
| 后端 | 自研 API 服务（Node + Hono + SQLite） |
| 配套 | 官网（Web）、管理后台（Admin）、自动更新服务 |

## 二、核心功能

### 1. AI 智能对话（Chat）
- 编辑器内置 AI 聊天面板，支持 **流式输出**（打字机式实时返回）。
- 结合当前文件、选中代码、工作区上下文进行问答、解释、重构、排错。
- 支持 **Mermaid 图表渲染**，AI 输出的流程图/时序图可直接在对话中可视化。
- 多会话管理，可保留历史会话。

### 2. 智能体模式（Agent）
- **Agent 模式**：把一个目标任务交给 AI，由其自主规划并多步执行。
- 支持 **工具调用（Tool Calling）**：AI 可读写文件、检索代码库、执行编辑等。
- **代码自动编辑与暂存**：AI 生成的改动以「待应用编辑」形式呈现，可逐项确认/回滚，重启后可恢复未完成的编辑。
- 后台任务（Agent Tasks）：长任务可在后台异步执行与追踪。

### 3. 行内代码补全（Inline / Tab 补全）
- 专用补全端点，基于光标前后文（prefix + suffix，FIM 思路）生成补全。
- 与聊天通道**独立限流**，保证补全低延迟、不互相抢占。
- 补全所用模型可独立配置，后续可无缝切换到专用 FIM 模型。

### 4. 代码库语义检索（Codebase Indexing / RAG）
- 对整个工作区建立**向量索引**，支持自然语言语义搜索代码。
- 接口：`embeddings`（向量化）、`upsert`（写入索引）、`search`（语义检索 Top-K）、按工作区清除。
- 让 AI 在回答与 Agent 执行时**理解整个项目**，而不仅是当前打开的文件。
- 索引按「用户 + 工作区 + 文件 + 分块」存储，相互隔离。

### 5. 多模型统一路由
- 一个下拉即可在不同厂商模型间切换，编辑器侧无需关心底层差异。
- 后端统一适配 **OpenAI 格式 / Anthropic 格式 / Vertex 格式**，对上层透明。
- 每个模型可单独配置：最大输入/输出 token、是否支持视觉、是否支持工具调用、计费倍率、启用与排序。

## 四、账号与计费体系

### 账号与登录
- **手机号 + 短信验证码（OTP）** 登录，集成短信通道。
- **JWT 鉴权**（默认 30 天有效），编辑器与官网之间通过 **SSO 单点登录**互通。
- 账号状态管理：封禁、注销（旧 token 立即失效）、登录 IP 记录、防滥用限流。
- 个人 **API Key** 管理，可用于程序化调用。

### 用量与计费
- 每次调用记录 **输入/输出 token、上游成本、计费金额（含倍率加成）、IP** 等。
- 支持 **按月用量上限**（`monthly_limit_usd`，可设为无限）。
- **套餐分级**：Free / Pro / Team。
- 集成 **Stripe** 支付与订阅（升级、订阅状态、计费周期）。

## 五、管理后台（Admin）

独立的 Web 管理后台（HTTP Basic 鉴权，与用户体系隔离），可视化运营：

- **模型管理**：增删改模型、定价倍率、能力开关、上下架、排序。
- **供应商管理**：配置各厂商接入地址、API Key、接口格式、自定义请求头。
- **用户管理**：查看用户、用量、套餐、封禁/解封。
- **用量统计**：成本与计费数据汇总。

## 六、技术架构

```
┌────────────────────────────────────────────────────────────┐
│  CodexaX 桌面客户端（VS Code 内核 + 自研 AI 扩展）            │
│   • codexax-chat   对话 / Agent / 代码编辑                    │
│   • codexax-auth   登录 / 账号 / 多模型 Provider             │
│   • theme-codexax  品牌主题   • mermaid 图表渲染             │
└───────────────┬────────────────────────────────────────────┘
                │  HTTPS（JWT / SSO）
┌───────────────▼────────────────────────────────────────────┐
│  CodexaX 后端 API（Node + Hono + SQLite）                    │
│   /chat  /completions  /indexing  /agents  /usage           │
│   /auth  /keys  /me  /billing  /admin  /update              │
│   多模型路由：OpenAI / Anthropic / Vertex 适配               │
└───────────────┬───────────────────────┬────────────────────┘
                │                       │
        ┌───────▼───────┐       ┌───────▼───────┐
        │ 官网 Web(Next)│       │ 管理后台 Admin │
        └───────────────┘       └───────────────┘
```

- **客户端**：VS Code（code-oss）+ 自研扩展（chat / auth / 主题 / mermaid）。
- **后端**：Node.js + Hono 框架 + better-sqlite3（轻量、易部署，可平滑升级到 Postgres/pgvector）。
- **官网 / 后台**：Next.js。
- **鉴权**：JWT + SSO；管理后台独立 Basic 鉴权。
- **数据**：用户、模型、供应商、用量、索引分块、Agent 任务等统一存储。

## 七、部署与自动更新

- **自动更新服务**：内置 `/update/:platform/:quality/:commit` 与 `/download` 接口，客户端可自动检测并升级。
- **多平台分发**：支持 macOS / Windows / Linux 客户端发布。
- **发布流程**：内置 releases 发布接口，可对接 CI 自动出包、发版。
- **私有化部署**：后端 + 官网 + 后台可整体部署在自有服务器，模型密钥与数据完全自控。
